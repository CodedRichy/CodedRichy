## ❖ CORE PROJECTS

> Selected engineering projects. Some repositories are private; descriptions focus on architecture and technical guarantees.

<br>

### ↳ **Expenso** — *Ledger-Based Group Expense System*
A cross-platform group expense application designed around strict accounting invariants.

**Key Engineering Decisions:**
> - Append-only expense revisions (no destructive updates)
> - Integer minor-unit accounting to eliminate rounding errors
> - Deterministic settlement computation (zero-sum invariant enforced)
> - Natural-language expense parsing using Groq/Llama
> - Direct UPI intent-based settlement flows

`Stack: Flutter, Firebase, Groq`

---

### ↳ **RoadPack** — *Real-Time Convoy Telemetry Platform*
A live location and coordination system for group rides and convoys.

**Technical Highlights:**
> - Real-time location synchronization
> - Google Maps telemetry integration
> - Group state consistency under network jitter
> - Optimized Firebase read/write patterns

`Stack: Flutter, Firebase, Google Maps Platform`

---

### ↳ **GitPulse** — *Autonomous Git Synchronization Daemon*
A background Python service that automates Git workflows with safety guarantees.

**Core Mechanisms:**
> - File-system event debouncing per repository
> - AI-generated commit summaries from diff context
> - Guardrails to prevent noisy or unsafe commits
> - Designed for long-running background execution

`Stack: Python`

---

### ↳ **RepoPilot** — *Codebase Intelligence & Automation Pipeline*
Infrastructure tooling for managing and enforcing structure across large repositories.

**Responsibilities:**

> - Pipeline-oriented execution model

`Stack: Python, Linux`
